## 1. 产品概述

自驾俱乐部路书编排台是专为车队领队设计的协同工具，解决多人车队出行前反复沟通、信息分散的痛点。领队通过三个核心页面完成活动创建、成员信息收集和路书发布，实现高效、专业的车队出行组织。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用场景 | 核心权限 |
|------|----------|----------|
| 领队 | 活动组织者，负责路线规划和发布 | 创建活动、编辑路书、管理成员、发布路书 |
| 车主 | 参与活动的驾驶员 | 提交报名信息、查看个人路书 |

### 2.2 功能模块

1. **活动设置页**：创建活动、填写基本信息、设置打卡点、自动生成节点时间线
2. **成员确认页**：收集车主信息、智能分析续航/经验、休息点建议
3. **车队路书页**：生成统一路书、分配车号、对讲频道、跟车规则

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 活动设置页 | 活动信息表单 | 活动名称、日期、集合点、车辆数、预计车速、住宿预算录入 |
| 活动设置页 | 打卡点管理 | 添加/删除必须到达的打卡点，支持排序和备注 |
| 活动设置页 | 节点时间轴 | 自动拆分为集合、午餐、补给、景区、住宿等节点，显示每段出发/到达时间窗口 |
| 成员确认页 | 报名列表 | 展示已报名车主及其详细信息 |
| 成员确认页 | 添加成员 | 录入车型、续航里程、驾驶经验、是否带老人孩子、是否愿意当尾车 |
| 成员确认页 | 智能分析 | 根据最弱续航和驾驶经验，提示是否需要增加休息点及补给建议 |
| 车队路书页 | 路书总览 | 领队视角的完整路书，包含车队编组、行程概览 |
| 车队路书页 | 个人路书卡片 | 每位成员的专属路书，含车号、对讲频道、跟车规则、迟到处理 |
| 车队路书页 | 发布功能 | 一键发布路书，生成分享链接 |

## 3. 核心流程

领队在活动设置页创建活动并填写基本信息 → 系统根据车速和打卡点自动生成节点时间轴 → 领队在成员确认页收集/录入车主信息 → 系统分析最弱续航和驾驶经验并给出休息点建议 → 领队在车队路书页调整编组和规则 → 发布路书，成员可查看个人专属路书。

## 4. 界面设计

### 4.1 设计风格

- **主色调**：深青色 #0F4C5C（专业、可靠），辅以琥珀橙 #E36414（警示、活力）
- **按钮风格**：直角转小圆角，强调专业感，主按钮使用深青色配白色文字
- **字体**：标题使用「思源黑体 Bold」，正文使用「思源黑体 Regular」，数字和时间使用等宽字体增强可读性
- **布局风格**：卡片式布局，左侧导航 + 右侧主内容区，信息密度适中
- **图标风格**：线性图标，2px 线条，保持简洁专业

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 活动设置页 | 活动信息表单 | 分组卡片、标签式输入、数值步进器 |
| 活动设置页 | 节点时间轴 | 垂直时间线、节点卡片、时间标签、可拖拽排序 |
| 成员确认页 | 报名列表 | 表格 + 卡片混合视图、状态标签、筛选器 |
| 成员确认页 | 智能分析面板 | 风险等级指示条、建议卡片、续航进度条 |
| 车队路书页 | 路书总览 | 分栏布局、车队编组列表、行程概览卡片 |
| 车队路书页 | 个人路书卡片 | 模拟纸质路书样式、车号徽标、对讲频道徽章 |

### 4.3 响应式

桌面端优先设计，平板自适应，移动端单列堆叠。核心操作在移动端保持可用。

### 4.4 动效设计

- 页面切换：淡入 + 轻微上移
- 节点生成：时间轴节点依次出现
- 表单校验：错误提示轻微抖动
- 数据加载：骨架屏占位
